# mCloud

A self hosted storage system.

## Install instructions:

```
cd #
sudo apt update && sudo apt install git -y
git clone https://github.com/beans-are-gross/mcloud.git
cd mcloud
sudo bash install.sh
```

## Update instructions:

```
cd #
rm -r mcloud
git clone https://github.com/beans-are-gross/mcloud.git
cd mcloud
sudo bash update.sh
```
